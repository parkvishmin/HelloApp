# HelloApp
Hello App is a progressive Java console application used to demonstrate
software development practices step by step.

The application evolves through multiple use cases:

1. Print Hello World
2. Accept user name from command line
3. Handle optional arguments
4. Process multiple names
5. Read names from standard input
6. Store names in collections
7. List and remove names
8. Refactor into classes
9. Add persistence across runs
10. Display greetings in banner format

## How to Run

Compile:

javac src/main/java/HelloApp.java

Run:

java -cp src/main/java HelloApp
